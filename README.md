## 练习方法

学一个知识点，就开始刷这个知识点的题

### 刷题

每一题做 2 遍

1. 第一遍，提交

2. 第二遍
   1. 和最高投票的题，去比较。学习别人的答案
   2. 尽量做到bugfree
   3. 提高编程速度

### 笔记和复习

1. 要写清楚考点，思考
2. 复习，每天花5分钟将前一天学的复习下
