# 1.3 HTML及相关格式 注释

### 注释的分类

一、 HTML 中的注释

```
<div>
      这是一行文字
      <!-- 这是一行被注释的文字 -->
</div>
```

二、CSS 中的注释

- 在 .html 文件中

```
<style>
  div {
      /* color: #fff;  */
   }
</style>
```

- 在 .css 文件中

```
div {
    /* color: #fff;  */
}
```

- 在 .less 或 .scss 文件中

```
div {
    /* color: #fff;*/  /* 多行注释*/
    // font-size: 14px; // 单行注释
    background: #000;
}
```
